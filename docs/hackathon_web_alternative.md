# STOX (ストックス)

**AIが「在庫」と「献立」をいい感じに管理する、ありそうでなかった買い物・料理エージェント**

*(デモ動画はこちらをクリック)*

## プロジェクト概要

STOXは、毎日の「料理」と「買い物」における「面倒くさい」を解消するFlutter製モバイルアプリケーションです。
Gemini Flashを活用し、冷蔵庫の中身やレシートを撮影するだけで、在庫管理から献立提案までをシームレスに行います。

### 主な機能

1.  **冷蔵庫内の食材認識**: 冷蔵庫を撮影するだけでGeminiが食材を解析・登録。
2.  **AIレシート解析**: レシートを撮影するだけで品目・数量・金額を自動登録。
3.  **インテリジェント・レシピ解析**: Web上のレシピを取り込み、自宅の在庫と照合して「作れるもの」「買い足すべきもの」を判断。
4.  **オンデバイスデータベース**: オフラインファーストで動作し、快適な使い心地を実現。

## Web版アプリについて

本プロダクトは、Flutterで開発されていますが、現在 **Androidアプリ (APK)** としての提供を主としており、Web版の公開は行っておりません。
ハッカソン審査においては、以下の理由によりWebブラウザでの完全な機能提供が困難であるため、本ページにて代えさせていただきます。

### Web版提供が困難な理由

1.  **オンデバイスMLの利用**
    レシートスキャン機能において、Google ML Kit Document Scanner および Camera プラグインを使用しており、これらはモバイルネイティブ（Android/iOS）の機能に強く依存しています。Webブラウザ（Wasm/JS）では同等のUX（歪み補正、鮮明化、高速な読み取り）を実現することが技術的に困難です。

2.  **ローカルデータベース (SQLite) の採用**
    ユーザーのプライバシーとオフラインでの応答速度を最優先するため、データ永続化には `Drift` (SQLite) を使用したローカルファーストアーキテクチャを採用しています。Web版で同様の挙動を実現するには、ブラウザ固有のファイルシステム制約やデータベース同期の仕組みを再設計する必要があり、短期間での実装が困難です。

本アプリのコア価値である「撮影してすぐ解析」「サクサク動く在庫管理」を体験いただくには、ネイティブアプリとしての動作が不可欠であると考えています。
